End-to-End Booking Flow

Happy Path Sequence

CLIENT → GATEWAY → MICROSERVICES

Live Transaction Flow

Process Execution Steps

1 Initiation

Customer submits booking request via API Gateway.

2 Creation

Booking Svc creates record & publishes seat-lock-request.

3 Locking

Seat Lock Svc locks in Redis & emits seat-locked.

4 Confirmation

Booking Svc confirms lock & publishes booking-created.

5 Payment

Payment Svc processes charge & emits payment-success.

6 Issuance

Ticket Svc generates ticket & publishes ticket-issued.

7 Notification

Notification Svc sends email/SMS confirmation to user, completing the saga.
