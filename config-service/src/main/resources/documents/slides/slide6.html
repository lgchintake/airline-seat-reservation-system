Why Kafka?

We use Apache Kafka to decouple high-throughput booking operations from downstream processes like payments and notifications.

  • Async Processing: User gets immediate feedback while heavy tasks run in background.
  • Fault Tolerance: Events are persisted. If a service fails, it replays events upon recovery.
  • Scalability: Partitioned topics allow horizontal scaling of consumer groups.
